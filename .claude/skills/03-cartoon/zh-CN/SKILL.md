# Seedance 2.0（Higgsfield）——卡通与动画提示词生成器

欢迎使用终极指南，为 Seedance 2.0（Higgsfield）创建令人惊艳的卡通和动画风格视频提示词。无论您是制作异想天开的角色动画、充满活力的动作序列还是极简主义的运动图形，此技能提供了框架、模板和深度创意指导，用于生成专注于动画的视频提示词。

## Seedance 2.0（Higgsfield）规格

**Seedance 2.0（Higgsfield）** 是一个针对创建高质量风格化视频内容优化的生成AI视频平台。对于动画工作流，请考虑这些核心规格：

- **原生帧数**：24fps、30fps、48fps、60fps（最适合动画；24fps = 经典剧院感）
- **时长甜蜜点**：2-6秒（比更长序列具有更高的每帧质量）
- **分辨率支持**：1080p、1440p、4K（推荐4K用于详细的手绘风格）
- **色彩深度**：完整8位RGB色彩空间（对于充满活力的卡通调色板至关重要）
- **运动向量**：Seedance 2.0（Higgsfield）支持平滑插值和运动模糊选项
- **提示词分词**：详细的风格描述符与动作描述符分开解析
- **迭代预算**：分别测试关键帧和风格变化以加快改进

## 卡通的2秒钩子框架

对于卡通和动画，前两秒应该**建立视觉风格和角色意图**。观众需要立即了解：

1. **美学类型**（这看起来像什么？迪士尼？日本动漫？平面设计？）
2. **角色或主体**（我在看谁或什么？）
3. **运动承诺**（接下来会发生什么？有能量吗？）

### 框架

```
[0–0.5秒]: [美学建立] + [主体展示] + [调色板]
[0.5–1秒]: [第一个动作或表达] + [性格提示]
[1–2秒]: [能量或节奏承诺] + [运动类型]
```

### 例子1：迪士尼风格

```
0–0.5秒：一个友好的棕色兔子，大眼睛，长耳朵。明亮、充满活力的颜色。柔软的线条。
0.5–1秒：兔子转身。大的、富有表现力的眼睛眨眼。微笑。
1–2秒：快速跳跃。弹性动画。运动线。高能量。
```

### 例子2：日本动漫风格

```
0–1秒：一个女孩，大的夸张眼睛，长黑发。赛璐珞着色。高对比度的轮廓。
1–2秒：爆炸式的表达变化。闪光和强调线。分布式背景。
```

### 例子3：扁平设计/极简

```
0–1秒：几何形状。渐变。大而清晰的色块。无轮廓。
1–2秒：形状平滑变换和移动。极小主义运动。
```

## 动画美学架构

### 第1层：定义您的卡通风格

#### 经典动画风格

| 风格 | 外观 | 传达的能量 | 常见应用 |
|------|------|----------|--------|
| **迪士尼古典** | 圆润的线条、大眼睛、拟人化动物 | 温暖、友好、可爱 | 儿童内容、角色动画 |
| **日本动漫** | 夸张的表达、大眼睛、高对比度轮廓、分布式背景 | 戏剧性、强烈、有男子气概 | 动作、冒险、浪漫 |
| **欧洲艺术** | 粗糙笔画、不规则线条、成熟的调色板 | 复杂的、知识分子的、古怪的 | 独立电影、文学改编 |
| **赛璐珞着色** | 平坦的颜色、清晰的黑色轮廓、最少的着色 | 电影化的、精美的、经典的 | 动作电影、大制作 |
| **橡皮软管** | 柔软、流体线条、夸张的身体变形 | 卡通化的、幽默的、复古的 | 喜剧、复古、实验 |
| **扁平设计** | 几何形状、没有轮廓、渐变、设计驱动 | 现代、简洁、专业的 | 品牌动画、UI、广告 |
| **3D卡通** | 赛璐珞着色应用于3D模型 | 可爱、现代、可接近的 | 儿童节目、广告 |

### 第2层：线条工作和轮廓

#### 线条风格定义

```
粗线条（3-5像素）→
- 外观：大胆、表现力强、儿童友好
- 使用场景：儿童节目、喜剧、可爱的角色
- 情绪：温暖、可接近、有趣

精细线条（1-2像素）→
- 外观：优雅、精致、工艺感
- 使用场景：艺术动画、长篇电影、复杂的角色
- 情绪：复杂、成熟、有意的

无轮廓（赛璐珞）→
- 外观：干净、现代、平坦
- 使用场景：品牌动画、3D卡通、极简主义
- 情绪：聚焦、现代、专业

可变线条（粗细变化）→
- 外观：艺术感、手绘、有机
- 使用场景：艺术电影、实验动画
- 情绪：创意、独特、手工制作
```

#### 轮廓色彩心理学

```
纯黑色 → 高对比度、高能量、戏剧性
深灰色（#333） → 柔和的轮廓、不那么严厉
棕色或焦土色 → 温暖、友好、复古
深蓝色 → 平静、优雅、夜间设置
```

### 第3层：调色板和色彩理论

#### 调色板策略

**高能量（儿童/喜剧）**
```
原色：红色、黄色、蓝色
附加颜色：绿色、紫色、橙色
饱和度：高（80-100%）
对比度：强烈
```

**温暖和舒适（故事讲述）**
```
主色：暖橙色、棕色、米色
辅助色：焦土色、金色、深红色
饱和度：中等（60-80%）
对比度：柔和
```

**冷和神秘**
```
主色：深蓝色、深紫色、青色
辅助色：薄荷绿、银色、淡紫色
饱和度：中等到高
对比度：对比强
```

**现代和极简**
```
主色：中立（灰色、白色、黑色）
口音：一个或两个非常明亮的颜色
饱和度：低到中等（除了口音）
对比度：精确和意图
```

#### 颜色心理学

```
红色 → 能量、紧急、激情。用于动作、重要时刻。
蓝色 → 冷静、信任、悲伤。用于深思熟虑的场景。
黄色 → 快乐、警告、友好。用于喜剧、日光。
绿色 → 自然、增长、毒性。用于环境或外星人设置。
紫色 → 魔法、神秘、皇家。用于幻想或高端场景。
橙色 → 温暖、友好、兴奋。用于日光和快乐的时刻。
灰色 → 中立、悲伤、未来派。用于城市或科幻设置。
```

### 第4层：动画风格和运动类型

#### 动画类型

| 类型 | 特征 | 运动感觉 | 用途 |
|------|------|--------|------|
| **完全动画（12fps）** | 每帧都绘制，平滑 | 液体、逼真的 | 电影、复杂动作 |
| **有限动画（8fps）** | 关键帧之间的跳跃 | 风格化、节奏化 | 电视动画、成本控制 |
| **单帧动画（4fps）** | 清晰的关键帧图像 | 卡通、卡通化 | 角色切换、演讲 |
| **运动图形** | 文本和形状移动 | 设计驱动、流畅 | 品牌、解释器、广告 |
| **停止运动** | 逐帧移动拍摄 | 物理、纹理、有机 | 超现实、特色、工艺感 |

#### 运动风格描述符

```
弹性运动 → 额外往返，超射。充满活力，可爱。
流体运动 → 长缓冲，弧形。优雅，逼真。
僵硬运动 → 快速关键帧转变。机械，卡通化。
漂浮运动 → 缓慢加速和减速。梦幻，空灵。
冲击运动 → 快速启动，立即停止。冲击，能量。
摇晃运动 → 小的高频抖动。紧张，焦虑。
```

### 第5层：背景和环境

#### 背景风格

```
极简背景 → 纯色或简单渐变。焦点在角色上。
线条背景 → 黑色线条定义形状。廉价，风格化。
分布式背景 → 速度线、动作背景、拉伸背景。日本动漫。
详细环境 → 完整绘制的背景。电影、工作室质量。
3D背景 → 实时3D与2D角色叠加。现代、混合。
数据可视化 → 网格、数字、代码雨。科技、未来派。
```

### 第6层：VFX和视觉效果

#### 动画VFX元素

```
速度线 → 运动指示、能量、动作感
拖尾 → 脱落的剪影显示运动弧
闪光 → 对特别重要的动作的强调
粒子 → 灰尘、星星、雪、光
形状变形 → 变形、拉伸、扭曲以显示力
光或发光 → 魔法、能量、重要时刻
对焦效果 → 模糊的背景，锐利的前景
```

## 完整提示词模板

### 标准卡通角色动画

```
[角色]: [名称/描述]。[风格]（例如迪士尼、日本动漫、扁平设计）
[外观]: [头发、眼睛、身体、服装描述]
[调色板]: 主色：[颜色]。次要颜色：[颜色]。口音：[颜色]。
[线条工作]: 线条粗细：[粗/细/可变]。轮廓颜色：[颜色]。
[背景]: [极简/分布式/环境]
[打光]: [简单的平面着色]或[柔和的阴影]
[动画风格]: [完全/有限/单帧]。运动：[弹性/流体/僵硬]
[动作]: [0–2秒动作描述]
[音频]: [配乐类型]或[语音同步]
[VFX]: [速度线]、[闪光]等
```

### 标准日本动漫风格

```
[场景]: [设置]。[时间]。
[角色]: [名称]。[年龄]。[个性]。大眼睛，长头发。赛璐珞着色。
[动画]: 有限动画（8fps）。闪光。分布式背景。表达变化爆炸式。
[颜色]: 高对比度。高饱和度。[主色]和[次色]。
[打光]: 高键。清晰的阴影区域。高对比度。
[背景]: 分布式或极简。速度线。
[声音]: 动漫配乐。可能的配音。
[动作]: [0–3秒动作，通常是关键帧式的]
```

### 标准扁平设计动画

```
[概念]: [描述]
[风格]: 扁平设计。几何形状。无轮廓。渐变。
[颜色]: 调色板：[5种颜色列表]。中等饱和度。
[类型]: 运动图形、品牌动画或故事讲述。
[形状]: 简单的几何或有机形状。平滑边缘。
[动画]: 平滑变换。形状变形。[1-2秒到位]
[背景]: 单色或简单渐变。
[音乐]: 现代、极简的配乐或上升音。
[文本]: [如果适用，无衬线字体、简洁消息]
[VFX]: 最少。可能的闪光或淡出。
```

### 标准儿童故事动画

```
[故事]: [叙述]
[角色]: [友好的、可爱的设计]。[和蔼的颜色]。
[动画]: 完全动画（12fps）。流体和富有表现力的运动。
[背景]: 详细和温暖的环境。配色精心选择。
[调色板]: 温暖颜色。中等到高饱和度。柔和的对比度。
[配音]: 友好的叙述声音。音乐。SFX（动物声音等）
[运动]: 弹性、可爱、友好的运动风格。
[VFX]: 闪光、粒子（星星、魔法）。柔软的过渡。
[节奏]: 缓慢到中等。清楚的故事弧。
```

## 高级优化

### 1. 同步到音乐或配乐

```
提示词片段：
"动画同步到音乐节拍。
击鼓 = 角色跳跃或重击
弦乐高潮 = 表达变化或特写
低音鼓 = 角色落地或冲击
```

### 2. 角色设计系统

对于系列或多角色项目：

```
主要角色：[高详细度，突出功能]
次要角色：[简化版本，可识别]
群众：[再次简化，一致的风格]

保持一致的：
- 线条粗细和风格
- 眼睛形状和大小
- 颜色关系
- 人体比例
```

### 3. 连续性和一致性

```
如果生成多个镜头：
- 保持相同的角色模型（使用@material参考）
- 保持相同的背景风格
- 保持相同的动画风格和帧率
- 保持相同的色彩分级
```

### 4. VFX强调时刻

```
高能量时刻：
- 速度线
- 闪光
- 增加的饱和度
- 更快的帧率（12fps升至24fps）
- 运动模糊

情感时刻：
- 过度曝光或淡化
- 放缓（6fps）
- 闪光或光芒
- 颜色饱和度下降（悲伤）
```

## 常见陷阱

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 看起来生硬 | 缺乏缓冲和弧形 | 添加"流体缓冲进出" |
| 调色板不一致 | 过多的随机颜色 | 选择3-5种互补色 |
| 没有能量感 | 运动太缓慢或太线性 | 使用弹性或冲击运动 |
| 背景分散注意力 | 背景和角色同等复杂 | 简化背景；降低对比度 |
| 看起来太3D | 光线和阴影过于逼真 | 使用平坦着色；移除复杂的阴影 |
| 无法识别的字符 | 风格不一致 | 保持线条粗细和眼睛风格一致 |
