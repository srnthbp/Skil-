## Seedance 2.0（Higgsfield）接受的内容

**输入规格：**
- **图像：** 最多9张图像（参考资料、故事板、情绪板、角色设计、位置照片）
- **视频：** 最多3个视频片段（现有素材、转场、运动参考）
- **音频：** 最多3个音频文件（音乐轨道、音效、对白录音、环境音床）
- **文本：** 完整的提示词描述和规格说明
- **组合资源：** 所有类别中最多12个项目
- **输出时长：** 4-15秒的720p视频，带同步音频
- **特殊语法：** `@material[名称]`参考提示词中上传的参考资料

**输出特性：**
- 高质量720p视频，完整音频集成
- 与输入音频的精确时间同步
- 运动尊重提供的参考资料
- 顺序请求间的一致渲染

## 2秒钩子框架

电影级视频的成功取决于前两秒的吸引力。这个框架帮助您设计引人入胜的开场。

### 核心概念

**钩子 = 一个强有力的视觉承诺 + 独特的美学声明**

前两秒应该清楚地传达：
1. **视觉类型**（这是什么类型的图像？）
2. **情感基调**（感觉如何？）
3. **运动承诺**（接下来会发生什么？）

### 2秒钩子框架模板

```
[MOMENT 0–1秒]: [视觉冲击] + [打光特征]
[MOMENT 1–2秒]: [摄像机运动] + [深度线索]
```

### 电影级钩子的例子

**例子1：黑色电影悬念**
```
MOMENT 0–0.5秒: 暗淡的红霓虹灯反射在湿润的街道。高对比度，多雾。
MOMENT 0.5–1秒: 特写镜头从眼睛拉出，揭示一张受伤的脸庞。柔软的焦点边缘。
MOMENT 1–2秒: 稳定摄像机平推右侧，深度雾化层。
```

**例子2：科幻史诗**
```
MOMENT 0–1秒: 一个宏大的未来城市天际线，极致景深，远处城市模糊。金色日落。
MOMENT 1–2秒: 吊臂镜头向上升起，揭示城市的规模。数字光线扫过摩天大楼。
```

**例子3：浪漫驾驶场景**
```
MOMENT 0–1秒: 一辆复古敞篷车从观众身边驶过，侧面对光。镜头光晕。
MOMENT 1–2秒: 切到车内，角色头转，金色小时光线透过窗户。景深浅。
```

## 电影级视频的语言架构

### 第1层：摄像机语言

#### 摄像机类型和运动

| 技术 | 定义 | 何时使用 |
|------|------|---------|
| **稳定摄像机推进** | 缓慢、流畅的向前或向后移动 | 戏剧性揭示，建立镜头，情感时刻 |
| **推车镜头** | 侧向滑动运动，通常跟踪主体 | 跟随角色，展示场景深度 |
| **吊臂镜头** | 向上或向下的弧形运动 | 建立全景，揭示环境规模 |
| **旋转** | 围绕轴旋转 | 揭示环境信息，戏剧性转折 |
| **变焦** | 焦距变化（光学或数字） | 强调重要元素，快速转向 |
| **空中无人机镜头** | 从高角度的宽视图 | 大规模场景，位置建立，追逐 |
| **过肩镜头** | 在主体肩膀上拍摄 | 对话、对抗、互动时刻 |
| **低角镜头** | 从下往上拍摄 | 权力、胁迫、威胁感 |
| **高角镜头** | 从上往下拍摄 | 弱点、脆弱性、孤立感 |

### 第2层：打光设计

#### 打光基调和风格

| 风格 | 特征 | 常见应用 |
|------|------|---------|
| **高键打光** | 明亮、均匀、最少阴影 | 希望、安全、乐观场景 |
| **低键打光** | 高对比度、戏剧性阴影、黑暗区域 | 神秘、危险、悬念、恐怖 |
| **电影比例** | 3:1至5:1的键盘对填充比例 | 标准电影外观，深度感 |
| **环境光/背光** | 从后面照亮主体的光 | 分离主体从背景，三维感 |
| **单色打光** | 仅使用一个关键光源 | 戏剧性、强调形式、风格化 |
| **多色打光** | 冷暖对比的多个光源 | 复杂的情感、动态场景 |
| **практический光** | 火焰、霓虹、灯光等源光 | 现实感、位置真实性 |

#### 打光颜色心理学

```
金色/琥珀色小时光线 → 温暖、怀旧、浪漫
冷蓝色 → 孤立、科幻、不安
红色/橙色 → 紧张、危险、激情
绿色 → 不自然、病态、科幻
单色黑白 → 永恒、经典、强调形式
```

### 第3层：色彩分级和心理学

#### 色彩调色板的战略使用

```
暖色调（金色、琥珀色、红色） →
- 使用场景: 日出、篝火、家内场景、 flashback
- 情感: 温暖、安全、怀旧、爱

冷色调（蓝色、青色、紫色） →
- 使用场景: 夜间、海水、科幻、死亡场景
- 情感: 孤立、神秘、不安、超越

脱色/褪色（低饱和度） →
- 使用场景: 战争电影、心理戏剧、记忆
- 情感: 绝望、厌倦、年龄感

高饱和度鲜艳 →
- 使用场景: 动作、冒险、喜剧
- 情感: 能量、兴奋、戏剧性
```

### 第4层：景深（DOF）和焦点

#### 景深策略

| 策略 | 视觉效果 | 情感影响 |
|------|---------|---------|
| **浅景深（f/1.4–f/2.8）** | 锐利主体，模糊背景 | 隔离、亲密、关注 |
| **中等景深（f/2.8–f/5.6）** | 主体和一些背景清晰 | 平衡的重点 |
| **深景深（f/8+）** | 所有内容清晰 | 环境意识、全景信息 |
| **移动焦点** | 焦点在拍摄过程中转移 | 揭示信息、引导注意力 |
| **光学光晕** | 背景光线漂亮失焦 | 电影质量、梦幻、浪漫 |

### 第5层：运动设计和节奏

#### 运动类型和意义

```
线性运动 → 目的、决定性、前进动力
抛物线运动 → 自然、流体、优雅
角运动 → 冲击、速度、紧张
缓冲运动 → 重量、权力、物理性
突发运动 → 惊吓、能量、冲击
```

### 示例1：黑色电影夜间场景

```
场景：下雨的城市街道，午夜。潮湿的水泥反射霓虹灯。

视觉设计：
摄像机：稳定摄像机推进右侧，跟踪角色
打光：单色关键灯从上方左侧。低键。对比5:1。深蓝色和红色霓虹灯。
焦点：浅景深（f/2.0）。焦点在角色眼睛上。

2–4秒：吊臂镜头向上，揭示空荡荡的街道。单个红色霓虹灯招牌。
4–6秒：稳定摄像机旋转，角色转身，戏剧性背光。

情感基调：神秘、危险、孤立。悬念。
```

## 高级提示词优化

### 1. 序列性和连续性

要链接多个"胶片场景"以创建连贯的故事：

```
[场景1 - 建立]: 摄像机宽拉。打光建立位置。
[场景2 - 对话]: 切换到过肩镜头。打光和色彩匹配场景1。
[场景3 - 行动]: 吊臂镜头。紧张的音乐。
[场景4 - 高潮]: 快速蒙太奇。明亮的闪光。高对比度。

确保色彩分级跨场景连贯。保持打光逻辑（日光源始终来自同一方向）。
```

### 2. 音频同步

```
prompt = "对话场景。角色说：'这不是你想的那样。'
打光切换到冷蓝色。摄像机缓慢推进。
同步音频节奏：每个句子的关键词前暂停500毫秒。"
```

### 3. VFX 和元素

```
prompt = "科幻操舱。屏幕上的绿色全息元素。
稳定摄像机旋转以跟踪全息图。
强调：全息图闪烁与对话同步。
打光：背光全息图，深蓝色和绿色。"
```

### 4. 特效光学效果

```
prompt = "城市天际线日落。添加镜头光晕效果（15度镜头）。
添加光晕：光点变模糊。
添加光学收差：远距红/青偏移在建筑物边缘。
打光：金色日光。背景模糊。前景尖锐。"
```

## 常见陷阱和修复

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 模糊的视觉 | 打光不清楚 | 指定关键光和填充光比例 |
| 平面感 | 缺乏景深 | 添加浅景深和背景模糊 |
| 不自然的颜色 | 过饱和 | 使用对比冷暖色的色彩分级 |
| 运动看起来急促 | 缓冲不足 | 指定"缓冲进出"和"流体运动" |
| 场景不一致 | 打光和色彩在镜头之间变化 | 创建"打光卡"，为整个序列保持一致 |
| 看不出是什么在发生 | 摄像机太静态 | 添加明显的摄像机运动以引导注意力 |

## 快速风格参考

### 好莱坞大制作

```
关键打光从45度角。清晰的背光。
景深浅（f/2.8）。
摄像机缓慢推进或稳定摄像机。
色彩分级：温暖的金色日光或冷蓝色夜晚。
```

### 电视纪录风格

```
自然打光。环境光。
景深深（f/5.6+）。
偶尔的手持运动。
饱和度较低的色彩分级。
```

### 艺术电影

```
不对称打光。大量阴影。
极端景深选择（非常浅或非常深）。
不寻常的摄像机角度和运动。
富有表现力的色彩分级或单色。
```

### 音乐视频

```
高度风格化的打光。
强大的色彩分级。
动态摄像机运动与音乐同步。
过度饱和或超现实的调色板。
```

## 最终提示

电影级胶片制作的秘诀不在于豪华或特殊效果——而在于**意图清晰**。每个灯光、镜头运动和色彩选择都应该为故事或情感服务。

使用这个框架作为你的创意工具包，而不是限制。最伟大的电影常常打破规则，但它们这样做是为了清晰的艺术目的。
